Grimple.gs

Dead simple, customizable, lightweight, cross-browser responsive grid system.

@version 0.0.0

View this project on Github

By @loup-brun

Why use Grimple?

Responsive

Grimple is responsive out of the box, using fluid proprotions to make your layout fit perfectly throughout any screen size. Which breakpoints? How many? How to call them? You chose.

Cross-browser

Grimple gives you peace in mind, as it does not rely on experimental technology while keeping your layout fluid down to IE6*. (if you want a complete grid with shiny flex, check out Reflex.gs)

Lightweight

Built on sass, you have full control on what to output and how many columns to include, without the need for heavy, bulky, frameworks that slow down page load.

* Remember than IE8 and below do not support media queries - if you really need those old browsers to follow your grid breakpoints at smaller sizes (which is unlikely to happen), you can include respond.js
